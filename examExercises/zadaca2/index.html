Преку javascript код да се имплементира опција за менување на стилот на параграфите во една страна. Кои стилови ќе може да се менуваат и со кои опции е дадено во еден json стринг. За секој од стиловите треба да се направи по еден select кој како опции ќе ги има опциите за дадениот стил. На почетокот параграфите немаат никаков стил и за дадениот json стринг треба да изгледа како на слика 1. Со менување на изборот на select се менуваа и соодветниот стил (слика 2, слика 3).
